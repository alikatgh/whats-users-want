# 11 — The headline findings

## Prerequisites

Modules [01-10](../README.md). This module is the **payoff**: how the
pipeline arrived at its four headline findings, with the chain of code
and statistics that supports each one.

## What you can do after

- Defend each finding in a presentation, with the supporting numbers and the
  code paths that produced them.
- Spot the same kind of analysis in another dataset and skip the dead ends
  the pipeline already hit.
- Recognise when a finding is robust ("three independent methods agree")
  versus when it's a single regression coefficient with a borderline p-value.
